# Expense Tracker Django

A web-based Expense Tracker built using Django that helps users manage budgets, track expenses, and monitor financial health.

## Features

- User Registration & Login
- Add, Edit, and Delete Expenses
- Category-wise Expense Tracking
- Monthly Budget Management
- Financial Health Dashboard
- Budget vs Expense Analysis
- Travel & Bills Pie Charts
- Food, Shopping & Other Bar Charts
- Recent Expense Tracking
- Search and Filter Expenses

## Technologies Used

- Python
- Django
- SQLite
- Bootstrap 5
- Chart.js
- HTML
- CSS
- JavaScript

## Installation

1. Clone the repository


git clone <repository-url>


2. Create virtual environment


python -m venv env


3. Activate virtual environment


env\Scripts\activate


4. Install dependencies


pip install -r requirements.txt


5. Run migrations


python manage.py migrate


6. Start server


python manage.py runserver


## Project Structure

- accounts/ → Authentication
- expenses/ → Expense Management
- templates/ → HTML Templates
- static/ → CSS, JS, Images
